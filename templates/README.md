# 🌿 Inner Echoes

## 📌 Description
Inner Echoes is a web application designed to help users express their thoughts and emotions through journaling. It provides a simple and interactive platform where users can reflect, write, and manage their inner feelings.

---

## 🚀 Features
- ✍️ Write and save personal journal entries
- 🔐 User-friendly interface
- 💡 Clean and simple design
- 🌐 Runs on a local server using Flask

---

## 🛠️ Technologies Used
- Python
- Flask
- HTML
- CSS

---

## ⚙️ Installation & Setup

### 1. Clone the repository
git clone https://github.com/your-username/your-repo-name.git

### 2. Navigate to the project folder
cd your-repo-name

### 3. Create a virtual environment (optional but recommended)
python -m venv venv

### 4. Activate the virtual environment
On Windows:
venv\Scripts\activate

On Mac/Linux:
source venv/bin/activate

### 5. Install dependencies
pip install -r requirements.txt

---

## ▶️ Run the Application
python app.py

After running, open your browser and go to:
http://127.0.0.1:5000
